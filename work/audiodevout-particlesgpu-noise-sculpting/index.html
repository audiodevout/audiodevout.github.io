Noise Sculpting with ParticlesGPU - Part 5

TouchDesigner Tutorials

In this project I used positional instancing through the particlesGPU tool, and used the default lookups to manipulate values and create some customisable 3d patterns! These are just some simple tricks to get you started with the particlesGPU tool which seemed very complicated to me in the beginning. Remember that TouchDesigner is extremely versatile so you can add all sorts of stuff in here and manipulate it as you wish!
