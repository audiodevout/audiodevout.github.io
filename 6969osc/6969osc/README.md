# 6969osc

Phone browser UI → WebSocket → Node on your PC → **OSC (UDP)** to TouchDesigner (default `127.0.0.1:6969`).

## Quick start (Windows)

1. Install [Node.js](https://nodejs.org/) LTS (v18+).
2. Copy `config.example.json` to `config.json` if needed; adjust ports/OSC host.
3. Double-click **`START.bat`** (run as Administrator the first time so the firewall rule can be added).
4. Scan the QR code in the terminal with your phone (same WiFi as the PC).
