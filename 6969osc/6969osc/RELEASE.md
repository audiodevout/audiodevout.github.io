# Publishing a 6969osc release

1. Zip the contents of this folder **without** `node_modules` (this folder is `6969osc/6969osc` in the repo; zip its files or delete `node_modules` from the archive after zipping).
2. Name the file something like `6969osc-v1.0.0.zip`.
3. On GitHub: **Releases → Draft a new release**, choose a tag (e.g. `6969osc-v1.0.0`), attach the zip, publish.
4. The site page [6969osc](https://audiodevout.github.io/6969osc/) links to **Latest release** for downloads.
