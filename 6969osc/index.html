6969osc

A browser-based control surface for your phone: buttons, toggles, XY pads, and a master strip. It talks to a small server on your PC over WebSocket and sends OSC (UDP) into TouchDesigner (or anything listening on the same machine).

This page only hosts documentation and downloads. At a venue you run START.bat on the show machine and scan the QR code so your phone loads the controller from your WiFi IP — not from GitHub Pages.

From the repository

Clone or browse the project, then install dependencies and start the server from the app folder.

  • Setup: npm install then npm start, or double-click START.bat on Windows.
